# Cakey-backend

This is backend repository for *Cakey app*

## Convention

[Google java style guide](https://google.github.io/styleguide/javaguide.html)를 따른다. \

## Docker compose

### Quick start guide

```bash
cp .env.example .env
./scripts/start.sh
```

### Scripts

#### Stop docker compose

```bash
./scripts/stop.sh
```

#### Run mariaDB CLI

```bash
./scripts/mariadb_cli.sh
```
